# Hi, I'm Kuan-Hao Chao

I am a Senior Deep Learning Scientist & Engineer at the [Illumina AI Lab](https://www.illumina.com/informatics/ai-in-genomics.html). I build machine learning methods and open-source tools for genomics, with a focus on DNA language models, RNA splicing, sequence-to-function modeling, and genome annotation.

For a fuller view of my research, publications, software, blog posts, and CV, visit [khchao.com](https://khchao.com).

[Website](https://khchao.com) · [Google Scholar](https://scholar.google.com/citations?user=n2AvFg0AAAAJ&hl=en) · [ORCID](https://orcid.org/0000-0003-0099-0692) · [LinkedIn](https://www.linkedin.com/in/kuanhao-chao/) · [Email](mailto:kuanhao.chao@gmail.com) · [Coffee chat](https://calendly.com/kuanhao-chao/30min)

## What I Build

- **DNA language models** for predicting gene regulation and expression from sequence.
- **Splice-site and variant-effect models** for cleaning RNA-seq alignments and interpreting cryptic splicing.
- **Open-source genomics software**: retrainable PyTorch frameworks, annotation lift-over tools, and graph algorithms for genome indexing.
