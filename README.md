# Hi, I'm Kuan-Hao Chao

I am a Senior Deep Learning Scientist & Engineer at the [Illumina AI Lab](https://www.illumina.com/informatics/ai-in-genomics.html). I build machine learning methods and open-source software for genomics, with work spanning DNA language models, RNA splicing and variant-effect prediction, sequence-to-function modeling, genome annotation, and pangenome algorithms.

I earned my Ph.D. in Computer Science from the [Center for Computational Biology](https://ccb.jhu.edu/) at Johns Hopkins University, advised by Steven Salzberg and Mihaela Pertea. My research and software include Shorkie, OpenSpliceAI, Splam, LiftOn, Han1, the Wheeler Graph Toolkit, and sangeranalyseR; for publications, software, blog posts, CV, and contact links, visit [khchao.com](https://khchao.com).
